# 第十一章 雲端服務平台的建立

## 11-1 理念
本章節將示範如何將智慧居家API與雲端服務API進行銜接，來完成本書物聯網程式的最終成品樣貌，並且最終成品擁有許多已經建立好的可擴展空間，例如可以輕鬆新增API呼叫方法進去，擴展整個物聯網的規模，而且擴展的過程中不需要去動到API的程式，只只須更新使用者介面的版本，若想要幫API進行加密也只需稍加修改程式即可，並不會對整體架構造成影響。
